**Problem being solved:** Every `huh.Run()` call in the codebase bypasses cobra I/O, making interactive commands untestable. `filter_prompt.go`, `adopt.go`, and `annotate_cmd.go` all import `huh` directly and call `os.Stdin`/`os.Stderr` raw.

**Solution:** Make `prompts.go` the single owner of all huh interactions. Key insight: `isTTY(cmd.InOrStdin())` returns false for `strings.Reader` (no `Fd()` method) → accessible mode triggers automatically → zero special test setup needed.
