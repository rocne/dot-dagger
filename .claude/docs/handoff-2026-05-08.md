## Key v2 design decisions to know

**Annotations:** `@key(args)` function-call style. `# @when(os=macos)`, `# @link(~/.tmux.conf)`, `# @after(base)`. Blank lines do NOT stop the scan; first non-comment, non-blank line stops.

**`.dagger` files:** Per-directory YAML config. Example:
```yaml
defaults:
  actions:
    - source
  when: "os=macos"
link_root: "~"
```

**`RawNode` fields:** `Path`, `LogicalName`, `EffectiveWhen`, `Actions []Action`, `After []string`, `LinkRoot`, `IsCompose`, `ComposeTarget`

**Logical name:** `node.DeriveName(relPath)` strips `nosync-`, `dot-`, extension from final component. `nosync-work/shellrc/aliases.sh` → `work.shellrc.aliases`.

**env.yaml:** Flat `map[string]string`, supports `$(dotd get-os)` shell expressions. No nested sections.

**config.yaml:** Tool preferences, no env keys. Fields: `dotfiles`, `bin_dir`, `generated_dir`, `link_root`.

**Versioning:** Staying at v0.x.x. Will cut v1.0.0 on stable release. No "v2" label anywhere.
